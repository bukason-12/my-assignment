100 HTML Tags and Their Uses

1. Document Structure Tags

<html> - Root element of HTML document

<head> - Contains metadata

<body> - Visible content

<title> - Page title

<meta> - Metadata

<link> - Links external files

<style> - Internal CSS

<script> - JavaScript code

<base> - Base URL

<!DOCTYPE> - Declares HTML version

2. Text Formatting Tags

<h1> - Main heading

<h2> - Sub heading

<h3> - Smaller heading

<h4> - Smaller heading

<h5> - Smaller heading

<h6> - Smallest heading

<p> - Paragraph

<br> - Line break

<hr> - Horizontal line

<b> - Bold text

<strong> - Important bold text

<i> - Italic text

<em> - Emphasized text

<u> - Underlined text

<mark> - Highlight text

<small> - Smaller text

<sub> - Subscript

<sup> - Superscript

<del> - Deleted text

<ins> - Inserted text

3. List Tags

<ul> - Unordered list

<ol> - Ordered list

<li> - List item

<dl> - Description list

<dt> - Term

<dd> - Description

4. Table Tags

<table> - Table

<tr> - Table row

<td> - Table cell

<th> - Table header

<thead> - Table head

<tbody> - Table body

<tfoot> - Table footer

<caption> - Table title

5. Form Tags

<form> - Form container

<input> - Input field

<textarea> - Multi-line input

<button> - Button

<select> - Dropdown

<option> - Option in dropdown

<label> - Label for input

<fieldset> - Groups form elements

<legend> - Title for fieldset

6. Media Tags

<img> - Image

<audio> - Audio

<video> - Video

<source> - Media source

<iframe> - Embed page

<canvas> - Drawing area

7. Semantic Tags

<header> - Page header

<footer> - Page footer

<nav> - Navigation

<section> - Section

<article> - Article

<aside> - Side content

<main> - Main content

<figure> - Image container

<figcaption> - Image caption

8. Interactive Tags

<details> - Expandable content

<summary> - Heading for details

<dialog> - Popup box

9. Programming/Embedded Tags

<script> - JavaScript

<noscript> - Fallback text

10. Miscellaneous Tags

<div> - Block container

<span> - Inline container

<time> - Time/date

<progress> - Progress bar

<meter> - Measurement
